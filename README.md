
# Lading Page

## Run Locally

- Clone the project

```bash
  git clone https://github.com/Abhineet-K/task_thence.git
```

- Go to the project directory



- Install dependencies

```bash
  npm install
```

- Start the server

```bash
  npm run dev
```

- Open http://localhost:5173 to view it in your browser.
## Tech Stack

**Client:** React, TailwindCSS, SwiperJS
